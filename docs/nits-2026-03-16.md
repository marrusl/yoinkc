## 3. Editor

1. **Read-only label unnecessary** — view mode is implied before clicking Edit
2. **Revert traps you in edit mode** — after editing then reverting, no way to
   exit editing. Saving the unchanged file marks it "not rendered" in file list
3. **File list has no structure** — needs categorization or at minimum
   alphabetical ordering
4. **Fleet variant confusion** — multiple copies from multiple machines in one
   flat list is confusing. Need smooth multi-variant expression in editor tab
5. **Edit doesn't activate re-render** — editing a file should light up the
   re-render button
